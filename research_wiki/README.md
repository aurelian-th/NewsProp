# NewsProp Research Wiki

This directory is the durable research memory for the autonomous loop. It exists so the implementation and the paper stay coupled.

## Files
- `phase_map.md`: what each phase does, what exists, and what is missing.
- `paper_outline.md`: evolving paper structure for the final submission.
- `experiment_log.md`: concise chronological record of experiments and outcomes.
- `assumptions.md`: model assumptions that need justification in the paper.
- `literature_backlog.md`: research questions and papers to collect later.

## Usage Rule
When a design choice, metric, calibration, or interpretation matters for the paper, document it here before moving on.
