## 🛠️ Technical Stack

## 📋 Specific Tasks
- [ ] **Graph Generation:** Generate a Barabasi-Albert scale-free network (e.g., $N=1000$ nodes, $m=3$ edges to attach from a new node to existing nodes). This mimics real social networks with "influencer" hubs.
- [ ] **Agent Profiling:** Write a script to assign attributes to each node $S_i$ based on realistic distributions (Normal/Uniform):
    * `age` (18-80)
    * `critical_thinking` (0.0 to 1.0)
    * `media_trust` (0.0 to 1.0)
    * `preferred_channel` (Mouth, Social, TV)
- [ ] **Centrality Calculation:** Calculate Betweenness and Eigenvector centrality for all nodes. Tag the top 5% as `hub_nodes`.
- [ ] Export graph object / adjacency matrix and agent dictionaries for Mesa.
