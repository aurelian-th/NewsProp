## 📈 Verified Result

On the current branch, the full default run against the 1,000-node Phase 3 payload produced:

- baseline peak infected: 778.6
- pre-bunked peak infected: 736.9
- peak reduction: 5.36 percent
- infected AUC reduction: 6.41 percent

This is enough to show a real intervention effect in a presentation, while still being small and reproducible enough to rerun quickly.
