# Phase 1: Data Collection & Scraping Layer

## 📝 Overview
Extract raw articles, engagement metrics, and fact-checks from Moldovan sources to feed the NLP layer. We need a mix of highly credible news and confirmed fake news.

* **Status:** To Do
* **Locked By:** None (Can start immediately)
* **Unlocks:** Phase 2 (AI/NLP Layer)

## 🛠️ Technical Stack & Targets
1. **Fact-Checks:** `stopfals.md` (Ground truth for fake news). Tool: `DrissionPage` or direct API requests.
2. **Mainstream News:** ProTV, Jurnal.md. Tool: `parse.bot` API wrapper.
3. **Social/Viral Spread:** Telegram (Moldovan political/news channels). Tool: `Telethon` (Official API).

### Canonical JSON Schema (Final)
The final machine-readable schema is now versioned in:
- `schemas/news_record.schema.json` (single record)
- `schemas/news_dataset.schema.json` (array dataset)

These files are the source of truth for all pipelines.

### Enforcement Rule (Mandatory)
Schema is enforced in code via `scraper/common/schema.py` using `enforce_final_schema(...)`.
Every scraper must pass records through this function before writing JSON.

### Debunk Segmentation Rule (stopfals.md)
`debunk_context` is **not** split by midpoint.
It is extracted using marker-based detection (e.g., "în realitate", "de fapt", "afirmația este falsă").
If no reliable debunk marker is found, the full content remains in `body_text` and `debunk_context` is set to `null`.
