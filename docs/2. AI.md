# Phase 2: AI & NLP Layer

## 📝 Overview
Convert raw text from Phase 1 into mathematical vectors (embeddings), reduce their dimensionality, group them into topics, and assign simulated "infectiousness" scores based on emotional intensity.

* **Status:** To Do
* **Locked By:** Phase 1 (Needs `raw_news_dataset.json`)
* **Unlocks:** Phase 4 (Simulation Layer)

## 🛠️ Technical Stack
* `sentence-transformers` (MiniLM or multilingual-MiniLM since data is in Romanian/Russian).
* `umap-learn` (Dimensionality reduction).
* `scikit-learn` (DBSCAN clustering).
* `nltk` or `textblob` (Sentiment analysis).

## 📋 Specific Tasks
- [ ] **Embedding:** Run the `headline` + `body_text` through the embedding model to get dense vectors.
- [ ] **Clustering:** Apply UMAP to reduce dimensions to 2D/3D, then run DBSCAN to find natural clusters (e.g., "Pensions", "Elections", "War").
- [ ] **Feature Extraction (The Magic):** Create a script that analyzes the `top_comments` and `engagement_metrics` to compute:
    * `controversy_score` (0.0 to 1.0) - based on comment volume vs. likes.
    * `emotional_intensity` (0.0 to 1.0) - based on NLP sentiment extremes in text.
    * `transmission_probability` ($\beta$) - Base probability an agent will share this.
- [ ] Export to `processed_news_clusters.json` (Appending these new metrics to the original data).
