# Phase 3: Population & Network Layer

## 📝 Overview
Build the mathematical graph representing Moldovan society and populate it with agents holding distinct psychological profiles.

* **Status:** Done
* **Locked By:** None (Can be built with dummy data immediately)
* **Unlocks:** Phase 4 (Simulation Layer)
* **Implementation:** `run_phase3_network.py` + `network/` module

## 🛠️ Technical Stack
* `networkx` (Graph generation and centrality metrics).
* `pandas` (Agent tables and exports).
* `numpy` (Distribution sampling and calibration).

## 📋 Specific Tasks
- [x] **Graph Generation:** Generate a Barabasi-Albert scale-free network (default $N=1000$, $m=3$) with deterministic seed support.
- [x] **Agent Profiling:** Assign realistic attributes to each node $S_i$:
    * `age` (18-80)
    * `critical_thinking` (0.0 to 1.0)
    * `media_trust` (0.0 to 1.0)
    * `preferred_channel` (Mouth, Social, TV)
- [x] **Centrality Calculation:** Compute Betweenness + Eigenvector centrality and tag top 5% as `hub_nodes`.
- [x] **Mesa Exports:** Export graph object / adjacency matrix and agent dictionaries for simulation.

## ▶️ Run Phase 3

From project root:

```bash
python run_phase3_network.py --nodes 1000 --m 3 --seed 20260477
```

Optional tuning:

```bash
python run_phase3_network.py --hub-fraction 0.05 --betweenness-k 256 --output-dir outputs/phase3_network
```

## 📦 Output Contract

The pipeline writes to `outputs/phase3_network/` by default:

* `moldova_network.graphml`
* `adjacency_matrix.csv.gz`
* `edge_list.csv`
* `agent_profiles.csv`
* `agent_profiles.json`
* `mesa_payload_phase3.json`
* `network_summary.json`

## 🧪 Notes

* Build is reproducible using `--seed`.
* For larger graphs, `--betweenness-k` can reduce runtime via sampled betweenness.
* Calibration source metadata is embedded in `network_summary.json`.
