# LaTeX Paper

This folder contains the Tectonic-ready manuscript source for the NewsProp paper.

## Files
- `NewsProp.tex`: full manuscript source

## Build Goal
Compile the paper with `tectonic` into a PDF without requiring a local TeX Live installation.

## Current Status
- The TeX source is ready.
- A local `tectonic` binary is not yet installed in this environment.
- The build wrapper is prepared so the paper can be compiled as soon as the binary becomes available.
