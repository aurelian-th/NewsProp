## Abstract
This paper presents NewsProp, a reproducible simulation framework for studying misinformation spread in a Moldova-inspired information environment. The system combines scraped fake-news, mainstream-news, and Telegram-style social content with a multilingual normalization pipeline, a synthetic scale-free social graph, and a SEIZ-style agent-based diffusion model. The current verified dataset pipeline processed 1,121 usable records after quality filtering. On a 1,000-node calibrated social network with 2,991 edges and 50 identified hub nodes, the baseline simulation produced a mean peak infected population of 886.3 agents with a 95% bootstrap confidence interval for the mean of [843.20, 929.01]. When a pre-bunking intervention was applied to hub nodes at the start of the simulation, peak infection dropped to 818.1 agents with a 95% bootstrap confidence interval of [784.39, 852.20], a reduction of 7.69%, while infected area under the curve decreased by 8.68%. These preliminary results suggest that targeted inoculation of influential nodes can materially reduce misinformation diffusion in synthetic Moldova-like networks.

## 1. Introduction
Online misinformation behaves like a contagious process: it spreads through social ties, adapts through resharing, and affects people differently depending on their trust, skepticism, and exposure patterns. Moldova is a particularly relevant context for studying this problem because its media ecosystem contains a mix of mainstream news consumption, social-media dependence, and politically charged narratives that move across multiple platforms.

The goal of NewsProp is to create a scientifically interpretable simulation rather than a generic demo. The project therefore joins data collection, multilingual preprocessing, synthetic population generation, and intervention testing into one reproducible workflow. The central research question is:

How does fake news propagate through a synthetic Moldovan information network, and to what extent can a targeted pre-bunking strategy reduce its spread?

## 2. System Overview
NewsProp is organized into five phases:

1. Data collection and scraping.
2. NLP normalization and impact scoring.
3. Synthetic network generation.
4. SEIZ-style propagation simulation.
5. Analysis, visualization, and paper generation.

This phase structure is important because the later claims depend on artifacts generated by earlier phases, not on hand-crafted examples.

## 3. Data Pipeline
The current repository includes scrapers and datasets for:

- fake news and debunked claims from StopFals and related sources
- real news from Moldovan mainstream sources such as `stiri.md`
- social-channel content from Telegram-focused scraping work

The Phase 2 normalization pipeline standardizes these sources into a common schema and computes downstream features such as:

- detected language
- English translation when enabled
- VADER sentiment
- emotional score
- emotional intensity
- engagement totals
- optional sentence-transformer embeddings

For the currently verified execution path used in this draft, Phase 2 was run in a robust smoke configuration with translation and embeddings disabled:

```powershell
.\.venv\Scripts\python.exe phase2\pipeline.py --disable-translation --disable-embeddings --output-dir phase2\outputs\smoke
```

This run kept 1,121 records and rejected 1 record during quality filtering.

## 4. Network Construction
Phase 3 generates a Moldova-inspired synthetic network using a Barabasi-Albert scale-free graph. Each node is assigned agent attributes such as:

- age
- critical thinking
- media trust
- preferred channel

Hub nodes are identified through centrality-based ranking. In the current verified 1,000-node run:

- nodes: 1,000
- edges: 2,991
- hub nodes: 50
- hub degree share: 24.86%
- average degree: 5.982
- degree Gini: 0.3796

These values support the intuition that a relatively small number of influential nodes account for a disproportionate share of network connectivity.

## 5. Simulation Method
Phase 4 uses a SEIZ-like model with the following states:

- `S`: Susceptible
- `E`: Exposed
- `I`: Infected
- `Z`: Skeptic

Agents interact across their graph edges and through preferred channels such as social diffusion and broadcast-like exposure. The simulator also includes a small distortion effect that perturbs news parameters during resharing. This approximates the way messages mutate as they spread.

An important engineering improvement completed in the current autonomous cycle is that the simulator now prefers Phase 2-normalized outputs when available. That means Phase 4/5 can directly consume:

- `impact_score`
- Phase 2 emotional intensity
- cleaned and normalized engagement metrics

instead of reconstructing all news features from raw scraped data.

## 6. Experimental Design
The primary verified experiment currently compares two scenarios:

1. Baseline spread.
2. Pre-bunking intervention where hub nodes are initialized as skeptics.

The verified full run used:

- 1,000 agents
- 10 runs per scenario
- 80 ticks
- initial infected seeds: 8
- hub fraction: 5%
- random seed: 42

The news item selected by the simulator in the verified full run was a fake-news article from `Facebook`, with Phase 2-derived properties:

- controversy score: 0.8225
- emotional intensity: 0.9965
- impact score: 0.9965
- transmission probability: 0.7748

## 7. Current Results
The verified full experiment produced the following summary:

| Metric | Baseline | Pre-bunked hubs | Relative change |
| --- | ---: | ---: | ---: |
| Peak infected | 886.3 | 818.1 | -7.69% |
| Peak infected 95% bootstrap CI | [843.20, 929.01] | [784.39, 852.20] | lower under intervention |
| Tick of peak | 6 | 7 | delayed by 1 tick |
| Infected AUC | 67,971.95 | 62,069.85 | -8.68% |
| Infected AUC 95% bootstrap CI | [64,735.38, 70,838.36] | [59,425.31, 64,719.04] | lower under intervention |
| Final infected | 886.3 | 818.1 | reduced |

These results indicate that targeting hub nodes before spread begins can both reduce the size of the peak and delay the diffusion wave. Even with the current heuristic and synthetic assumptions, the direction of the effect is stable and practically meaningful.

## 8. Interpretation
The most important current finding is not just that pre-bunking helps, but that it helps in a network-aware way. The intervention does not need to immunize the whole population. Instead, it reduces spread by focusing on the structurally influential part of the network. That aligns with the project’s intended thesis: misinformation control is more efficient when it is targeted rather than uniform. The newly added bootstrap intervals strengthen this claim by showing that the reduction is not only visible in point estimates, but also persists when run-to-run variability is summarized statistically.

## 9. Limitations
The present draft still has important limitations:

- the main verified run used a translation-disabled Phase 2 configuration for speed and robustness
- uncertainty intervals for intervention outcomes still need to be computed and reported
- the network is synthetic rather than directly estimated from Moldova-specific social graph data
- the current simulation parameters remain partially heuristic
- literature support and formal related-work citations are still pending

These are manageable limitations, but they must be addressed clearly before the paper is finalized.
